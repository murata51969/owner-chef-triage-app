トークンの作り方（初回のみ・スマホでOK）
1. GitHub のトークン作成ページを開く（要ログイン）
2. Token name: 任意（例 insider-review）／ Expiration: 1年など
3. Repository access: Only select repositories → 上記のデータリポジトリ（非公開）を選択
4. Permissions → Repository permissions → Contents: Read and write
5. Generate token → 表示された github_pat_... をコピーして上に貼り付け
※ トークンはこの端末（ブラウザ）内にのみ保存され、外部には送信されません。
※ データリポジトリには実名・投稿者IDなどの個人情報は含まれません（コメント本文と指標のみ）。
